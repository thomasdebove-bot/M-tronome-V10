# M-tronome-V10

Prototype front-end de home page avec:
- sélection d'une réunion,
- KPI des rappels ouverts et répartition par assigné,
- cumul des rappels/comptes-rendus par entreprise,
- vue calendrier des rendus filtrable par zone et lot.

## Lancer en local
```bash
python3 -m http.server 4173
```
Puis ouvrir `http://localhost:4173`.
